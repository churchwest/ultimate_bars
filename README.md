## Install Spec Kit

This repository includes Spec Kit as a dependency. You can install the tool locally
using one of the following methods.

Persistent install (recommended):

```bash
uv tool install specify-cli --from git+https://github.com/github/spec-kit.git
```

One-time usage with `uvx`:

```bash
uvx --from git+https://github.com/github/spec-kit.git specify init <PROJECT_NAME>
```

Install via pip (install the CLI package into your current Python environment):

```bash
pip install git+https://github.com/github/spec-kit.git
```

For more details see: https://github.com/github/spec-kit
